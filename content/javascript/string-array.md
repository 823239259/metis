# String - Array #

### array ###

+ 改变原数组

    + add

        + __push(...)__ (末位)

        + __unshift(...)__ (首位)

    + remove

        + __pop()__ (末位)

        + __shift()__ (首位)

    + splice

        + __splice(index, num, ...)__

    + sort

        + __reverse()__

        + __sort()__

+ 不改变原数组

    + __slice(start, end)__

    + __concat(...)__

    + __join(separator)__

    + slice

        + __substr(start, length)__

        + __substring(pos1, pos2)__

        + __slice(start, end)__

    + split

        + __split(separator, num)__

    + toggle

        + __toUpperCase()__

        + __toLowerCase()__
