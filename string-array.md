# String - Array #

### array ###
+ 改变原数组
    + add
        + __push(...)__ (last)
        + __unshift(...)__ (first)
    + remove
        + __pop()__ (last)
        + __shift()__ (first)
    + splice
        + __splice(index, num, ...)__
    + sort
        + __reverse()__
        + __sort()__
+ 不改变原数组
    + slice
        + __slice(start, end)__
    + concat
        + __concat(...)__
    + join
        + __join(separator)__
        
### string ###
+ 不改变原字符串
    + concat
        + __concat(...)__
    + slice
        + __substr(start, length)__
        + __substring(pos1, pos2)__
        + __slice(start, end)__
    + split
        + __split(separator, num)__
    + change
        + __toUpperCase()__
        + __toLowerCase()__
